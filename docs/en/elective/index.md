# Paper 2 · Electives Overview

> **Weighting:** 25% of the total subject mark · **Duration:** 1 hour 30 minutes · **Choose any 2 of 3 electives.**

In Paper 2 you sit a single booklet containing **three** sections — 2A, 2B and 2C — and answer **only the two** that correspond to the electives you have studied. Each elective contributes about 30 minutes of writing time and roughly the same number of marks.

## The three options side by side

| Aspect | **2A · Databases** | **2B · Web Application Development** | **2C · Algorithm & Programming** |
|--------|-------------------|--------------------------------------|----------------------------------|
| Curriculum hours | 38 | 38 | 38 |
| Main languages | SQL (SQL-92) | HTML, CSS, JavaScript, **PHP** (from 2026) | Python, C++ (Pascal until 2027) |
| Prerequisite Compulsory module | A · Information Processing | C · Internet & its Applications | D · Computational Thinking & Programming |
| Strengths required | Logical thinking, attention to syntax | Mix of design and coding | Algorithmic thinking, debugging |
| Typical difficulty | Medium | Medium | High |
| Predictability of questions | High | Medium | Lower (logic-heavy) |
| Best paired with | 2B (most popular) | 2A or 2C | 2A or 2B |

::: tip Which two should I take?
- **Most students choose 2A + 2B.** They share the most predictable mark schemes and rely on the same web-data toolchain (SQL + PHP + a database).
- **Maths-strong students** often add **2C** because algorithms questions reward clean logical thinking.
- **Avoid taking 2C alone with weak coding fundamentals** — Paper 2C tests algorithmic depth.
:::

## Common rules for Paper 2

- Questions test learning outcomes from your two chosen electives.
- Each elective consists of **short questions** and **structured questions**.
- All questions in each elective are compulsory — you may not pick and choose within an elective.
- The HKEAA provides a **reference sheet** with SQL commands, operators and ER diagram symbols inside Paper 2A.

## Programming language reference

| Year | Compulsory Module D | Elective 2B | Elective 2C |
|------|--------------------|-------------|-------------|
| 2025 DSE | Python (also accepts C++, Pascal) | JavaScript | Python (also accepts C++, Pascal) |
| 2026 DSE | Python (also accepts C++, Pascal) | **PHP** added | Python (also accepts C++, Pascal) |
| 2027 DSE | Python (also accepts C++, Pascal) | JavaScript + PHP | Python (also accepts C++, Pascal) |
| 2028 DSE → | **Only Python or C++** | JavaScript + PHP | **Only Python or C++** |

> Source: EDB "Curriculum Renewal on Senior Secondary ICT" handout and HKEAA reference papers.
