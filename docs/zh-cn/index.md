## 这个站点为谁而建？

HKDSE ICT 学习中心是一个免费的开源学习指南，紧贴**课程发展议会**与**香港考试及评核局**联合出版的官方《课程及评估指引》。它适合：

- **中四 - 中六学生**：想要一份覆盖所有学习要点的单一参考资料。
- **ICT 老师**：需要现成的教学计划、SBA 脚手架以及可信赖的三语参考资料。
- **自修生**：没有学校班级支援，独立备考公开考试。

## 站点内容

| 板块 | 你会获得什么 |
|------|------------|
| [必修部分](/zh-cn/compulsory/) | 卷一逐模块拆解，包含详细子话题、必备技能和示例题解。 |
| [选修部分](/zh-cn/elective/) | 三个选修的并排对比，加上 2A 数据库、2B 网络应用开发、2C 算法与编程的深度讲解。 |
| [SBA 指南](/zh-cn/sba/) | 任务一与任务二模板、评分标准解析、截图清单、主题灵感。 |
| [学习路线](/zh-cn/study-plan/) | 中四、中五、中六的每周时间表，以及给老师的专门策略。 |
| [应试策略](/zh-cn/exam-tips/) | 卷一/卷二时间策略、MC 陷阱分析、精选常见错误清单。 |
| [资源](/zh-cn/resources/) | 直接链接至官方 HKEAA / EDB 文件、历届试题、推荐工具及双语术语表。 |

::: tip 用最轻松的方式练习 SQL
卷二 2A 和你的 SBA 都需要写**大量 SQL**。不必本地安装 MySQL，可以直接在 **[SQL Books 在线编辑器](https://sqlbooks.codenav.dev)** 上练习 —— 每个账号都自带独立的沙盒数据库，让你随便测试 `SELECT`、`JOIN`、子查询等语句，完全不用担心搞坏。
:::

## 如何使用本站

1. **先从[必修部分](/zh-cn/compulsory/)开始**，复习卷一概念。
2. **选择你的两个[选修课题](/zh-cn/elective/)**，跟随专门章节学习。
3. **建立 SBA 任务一系统时收藏 [SBA 指南](/zh-cn/sba/)**。
4. **使用[资源](/zh-cn/resources/)页面的历届试题做模拟考试**。
5. **随时使用右上角语言切换器**，在英文、繁体中文和简体中文之间切换。

> **免责声明。** 本站是独立的学习辅助工具。课程大纲和考试规则的权威来源始终是 [HKEAA](https://www.hkeaa.edu.hk) 和 [EDB 课程发展处](https://www.edb.gov.hk)。如本站内容与任何官方文件冲突，请以官方文件为准。
