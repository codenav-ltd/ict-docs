## 为什么我们一直推荐 SQL Books

HKDSE 的 SQL 题靠**反复练习**形成熟练度。本地 MySQL 让你先与驱动、权限较劲，才写得到一句查询。SQL Books 去掉这层 —— 登录、粘贴、运行、看结果。每账号独立数据库，DROP TABLE 误操作不会牵连同学。

学校版还提供：

- 每生每项目独立数据库
- 内置 **SQL Books 课程**（basic → intermediate → advanced + DML）
- 教师面板含作业自动评分
- SBA 评审分享链接

试用：**[sqlbooks.codenav.dev](https://sqlbooks.codenav.dev)**。
