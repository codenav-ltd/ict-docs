# 3.2 · 变量与运算符

> **目标：** 声明变量、用对运算符、遵守命名规范。

## 变量

变量是给一个值起的名字。

```python
age = 17
name = "Alice"
height = 1.65
is_student = True
```

Python 推断类型 —— 无需显式声明。

## 命名规则

- 字母、数字、下划线；必须以字母或 `_` 起。
- 区分大小写：`Age` ≠ `age`。
- 别用 Python 关键字（`if`、`for`、`class`…）。
- **惯例**：变量与函数用 `snake_case`，常量用 `UPPER_CASE`。

好：

```python
average_height = 165
PI = 3.14159
```

坏：

```python
a = 165          # 'a' 是什么意思？
PiValue = 3.14   # 风格混杂
```

## 运算符速查

### 算术

| 运算符 | 含义 | 例子 | 结果 |
|----------|---------|---------|--------|
| `+` | 加 | `3+2` | 5 |
| `-` | 减 | `3-2` | 1 |
| `*` | 乘 | `3*2` | 6 |
| `/` | 真除 | `5/2` | 2.5 |
| `//` | 地板除 | `5//2` | 2 |
| `%` | 取余 | `5%2` | 1 |
| `**` | 幂 | `2**10` | 1024 |

### 比较（关系）

| 运算符 | 含义 |
|----------|---------|
| `==` | 等于 |
| `!=` | 不等 |
| `>` `<` | 大 / 小 |
| `>=` `<=` | 大等于 / 小等于 |

::: warning Python vs HKEAA 伪代码
HKEAA 伪代码用 `=` 比较、`←` 赋值。Python 用 `==` 比较、`=` 赋值。答案别混。
:::

### 布尔（逻辑）

| 运算符 | 含义 |
|----------|---------|
| `and` | 两个操作数都 True 时为 True |
| `or` | 至少一个 True 时为 True |
| `not` | 反转布尔 |

### 赋值快捷

| 快捷 | 含义 |
|----------|-------|
| `x += 5` | `x = x + 5` |
| `x -= 3` | `x = x - 3` |
| `x *= 2` | `x = x * 2` |

## 运算符优先级（高 → 低）

1. `**`
2. `*` `/` `//` `%`
3. `+` `-`
4. 比较 `<` `<=` `>` `>=` `==` `!=`
5. `not`
6. `and`
7. `or`

不确定就加括号。

## 实例

```python
total = 0
for i in range(1, 11):
    total += i              # 1+2+...+10
print(total)                # 55

a, b = 7, 3
print(a // b, a % b)        # 2 1   (商, 余)

x = 5
print(x ** 2)               # 25
print((x + 3) * 2)          # 16
```

## 学生常见错误

- 混淆 `=`（赋值）与 `==`（比较）。
- 忘了 `5 / 2 = 2.5`（Python 3 真除法）；整除用 `//`。
- 用 `&` 与 `|`（位运算）代替 `and` / `or`（布尔）。
- 用变量名遮蔽内置（`list`、`sum`、`str`）。

## 考试式题目

> **题（4 分）：** 逐行追踪 `result` 的值：
>
> ```python
> x = 7
> y = 3
> result = (x % y) * 2 + (x // y) ** 2
> ```

**参考答案：**

- `x % y = 1`
- `(1) * 2 = 2`
- `x // y = 2`
- `2 ** 2 = 4`
- `result = 2 + 4 = 6`

## 关键要点

- 名字重要 —— 取有意义的。
- 知道你的运算符及优先级。
- Python 的 `=` 是赋值；`==` 是比较。
