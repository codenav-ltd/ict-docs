# 3.1 · 版权基础

> **目标：** 定义版权并区分其与相关权利。

## 版权保护什么

**版权**是**原创**作品的创作者控制其**复制、发行、改编、表演与展示**的法律权利。

涵盖的作品：

- 文学作品（书、文章、源代码）
- 音乐作品
- 电影、电视节目、录制音频
- 美术作品（绘画、相片、图形）
- 建筑设计
- 计算机软件

版权**不**保护的：

- 想法（只保护想法的*表达*）
- 事实（二战日期）
- 政府法律与司法判决

## 版权如何取得

多数国家（含香港），作品一旦以有形形式创作即**自动**取得版权 —— 无需注册。© 符号与日期可选但是惯例。

## 期限

- 香港，文学、戏剧、音乐、美术作品版权一般为**作者寿命 + 50 年**。
- 期满后作品进入**公有领域**，可自由使用。

## 相关权利

| 权利 | 保护 |
|-------|----------|
| **商标** | 徽标、品牌 |
| **专利** | 发明（须新颖且实用） |
| **商业秘密** | 机密商业信息（如配方） |
| **设计权** | 工业设计（形状、外观） |
| **人格权** | 作者被署名与作品不被歪曲的权利 |

## 合理使用 / 公平处理

多数司法管辖区允许有限地使用版权作品无需许可，用于：

- 教育
- 新闻报道
- 评论 / 审查
- 戏仿 / 讽刺
- 研究

香港在《版权条例》(Cap. 528) 下用术语**公平处理 (fair dealing)**。

## 例子

| 行为 | 可能状态 |
|--------|---------------|
| 引用教材两句并附引用 | 公平处理 |
| 在电影院录电影并上传 | 侵权 |
| 未授权采样歌曲 | 通常侵权 |
| 分享名人 meme 图 | 常被容忍；法理复杂 |
| 按许可条款用开源库并署名 | 允许 |

## 香港法律背景

- **《版权条例》(Cap. 528)** 管港版权。
- **商业规模**侵权刑事责任（卖盗版 DVD）。
- 严重终端用户盗版可致罚款或监禁。

## 考试式题目

> **题（4 分）：** 解释版权保护什么并列三个涵盖作品的例子。

**参考答案：**

版权保护文学、音乐、美术与数字作品的**原创表达**，禁未经许可复制、发行、改编或表演。它不保护想法，只保护想法的表达方式。三例：出版的小说、录制的歌曲、计算机源代码。

## 关键要点

- 版权保护表达，不保护想法。
- 创作即自动；无需注册。
- 期限：通常作者寿命 + 50 年（香港）。
