# 2.2 · 人体工学

> **目标：** 描述符合人体工学的工作站并解释它如何减少损伤。

## 人体工学工作站清单

| 要素 | 建议 |
|---------|----------------|
| **椅** | 可调高度；腰部支撑；扶手；脚平放地面（或脚踏） |
| **显示器** | 屏顶在眼平；离眼 50–70 cm；抗眩光雾面 |
| **键盘** | 平或微倾；腕直，不上弯 |
| **鼠标** | 贴近键盘；手同高度；支撑腕垫 |
| **桌** | 稳定，够大放键鼠 + 文档 |
| **照明** | 间接；屏无反光；与自然光互补 |
| **空气** | 温度舒适；不太干 |
| **噪音** | 60 dB 以下；需要时降噪耳机 |

## 姿势图

```
  眼平 ------ 屏顶
              ↘
                ── 屏 50-70 cm
              
   肘 90°
   腕直
   背挺（微后倚 OK）
   髋齐 或略高于膝
   脚平放地面（或脚踏）
```

## 软件人体工学

课程指引提到**软件人体工学** —— 减少努力与挫败的设计：

- 跨屏**一致 UI**。
- **熟悉惯例**（`Ctrl-S` 保存）。
- **明确错误信息**。
- **无障碍**（屏幕阅读器支援、对比、字号缩放）。
- 常用动作**少点击**。

好软件人体工学 → 少出错、做得快、压力小。

## 常见坏习惯

- 笔电放腿上（颈太低、腕太高）。
- 单一大屏高度错且无独立键盘。
- 屏后亮窗 → 眼疲劳。
- 杂乱桌面强迫尴尬姿势。

## 忽视人体工学的代价

- 长期 RSI / 腕管可能要手术。
- 持续小痛降低生产力。
- 缺勤增多。

## 考试式题目

> **题（4 分）：** 为每天用笔电在饭桌上 4 小时上网课的学生建议四项人体工学改进。

**参考答案：**

1. 用支架**抬高笔电**并外接键鼠，让屏幕在眼平且手腕自然。
2. **用可调椅**带背支撑与脚踏，替换硬饭桌椅。
3. **改善照明** —— 把笔电与窗垂直放避眩光；晚学习用台灯。
4. **每 30–45 分钟休 5 分钟**站、伸展、跟 20-20-20 法则。

## 关键要点

- 调椅、屏、键、鼠、灯。
- 软件设计也影响人体工学。
- 小改动，长远健康收益大。
