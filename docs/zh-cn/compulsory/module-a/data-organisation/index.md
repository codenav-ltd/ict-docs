# 第 2 章 · 数据组织与控制

> **课时：** 4 · **难度：** ★★☆☆☆ · **衔接：** 选修 2A 数据库

数据只有在你能再次找到、并相信它没被破坏时才有用。本章教你原始数据如何**结构化**（组织）以及错误如何**预防与检测**（控制）。

## 学习成果

- 在数据的层级组织中**识别**数据、记录、字段、文件和数据库。
- **解释**记录如何被组织、储存和检索。
- **陈述**直接存取与顺序存取的优缺点和应用场景。
- **讨论**为何需要**数据控制**。
- **描述**错误如何透过 validation 和奇偶校验**检测**、透过 verification 和 validation **预防**。
