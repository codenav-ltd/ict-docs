# 4.1 · 电子表格基础

> **目标：** 说出电子表格各部分名称、写基本公式、用筛选 / 排序 / 搜索。

## 电子表格解剖

```
   A         B         C         D
1  Name      Class     Score     Grade
2  Alice     F.4A      86        =IF(...)
3  Bob       F.4A      72        =IF(...)
4  Carol     F.4B      91        =IF(...)
```

| 术语 | 含义 |
|------|---------|
| **Workbook 工作簿** | 整个文件（`.xlsx`） |
| **Worksheet 工作表（sheet / tab）** | 工作簿内的一页 |
| **Column 列** | 字母 A、B、C、… |
| **Row 行** | 数字 1、2、3、… |
| **Cell 单元格** | 一个交点（如 `B3`） |
| **Range 区域** | 矩形单元格组（如 `A2:D10`） |
| **Formula 公式** | 以 `=` 开头的表达式 |
| **Function 函数** | 预定义公式（如 `SUM`、`AVERAGE`） |

## 写你的第一个公式

每个公式都以 `=` 开始。运算符：

| 运算符 | 含义 | 例子 |
|----------|---------|---------|
| `+` `-` `*` `/` | 算术 | `=A1+B1` |
| `^` | 幂 | `=2^10` |
| `&` | 拼接文本 | `="Hello "&A1` |
| `=` `<>` `>` `<` `>=` `<=` | 比较 | `=A1>=50` |

```
A1: 50
A2: 30
A3: =A1+A2    →  80
A4: =A1*2     →  100
A5: =SUM(A1:A4) → 80+50+30+100 = 260
```

## 三个必备操作

### 1 · 排序

| 之前 | 按 Score 降序排后 |
|--------|---------------------------|
| Alice 86 | Carol 91 |
| Bob 72 | Alice 86 |
| Carol 91 | Bob 72 |

可按**单**或**多个标准**排（如先 Class 再 Score）。

### 2 · 筛选

只显示符合条件的行，其他行临时隐藏。

| 筛选 | 结果 |
|--------|--------|
| Class = F.4A | Alice 与 Bob 可见；Carol 隐藏 |
| Score ≥ 80 | Alice 与 Carol 可见 |

### 3 · 搜索 / 查找替换

- 查找：在工作表中定位一个值。
- 查找替换：批量把一个值换成另一个。

## 学生常见错误

- 公式开头忘了 `=`。
- 用 `*` 而不输单元格引用：`=A1 * 2` 可以，但 `=A1 * B1` 两个引用都要写。
- 排序时没选**整张表** —— 把名字和分数拆开。
- 不先清除之前的筛选就再筛 → 部分视图令人困惑。

## 实例 · 等级计算器

在 `D2` 输入：

```text
=IF(C2 >= 80, "A", IF(C2 >= 70, "B", IF(C2 >= 60, "C", "F")))
```

下拉到 `D3:D4` 即可给每位学生评级。

## 练习活动

新建上面所示的工作表。然后：

1. 按 score 降序排学生。
2. 筛只看 F.4A。
3. 加一列「Average」显示班级平均。
4. 用条件格式把低于 60 的分数标红。

## 关键要点

- 单元格、行、列、区域 —— 术语要熟。
- 每个公式以 `=` 开头。
- 排序、筛选、查找是第一天就要会的三个操作。
