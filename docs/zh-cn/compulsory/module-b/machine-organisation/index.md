## 全局图

```
          ┌────────── CPU ──────────┐
 Inputs ──│  Control + ALU + Regs   │── Outputs
          └────────┬────────────────┘
                   │ 总线
           ┌───────┴────────┐
           ▼                ▼
       ┌───────┐       ┌───────────┐
       │  RAM  │       │  Storage  │
       │  ROM  │       │  HDD/SSD  │
```

- 解释输入 / 输出设备、处理单元、总线系统、储存设备的功能。

- 概述 **fetch-decode-execute** 循环。
- 描述 RAM、ROM、缓存；将大小、地址、字长与性能联系起来。
- 按存取模式、易失性、传输速率、容量描述储存设备。
- 概述处理器、内存、储存、数据通讯的最新发展。
