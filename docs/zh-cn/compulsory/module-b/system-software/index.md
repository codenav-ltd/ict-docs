## 全局图

软件有两种：**系统软件**（让机器自身运转）和**应用软件**（用户真正用的）。

```
┌────────────────────────────┐
│ 应用软件                    │  Word、Chrome、游戏…
├────────────────────────────┤
│ 系统软件                    │  OS + 实用程序 + 驱动
├────────────────────────────┤
│ 硬件                        │  CPU、RAM、磁盘…
└────────────────────────────┘
```

系统软件让灯长亮。没有 OS、驱动、实用程序，应用软件跑不起来。
