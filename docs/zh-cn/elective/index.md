# 卷二 · 选修部分总览

> **比重：** 全科 25% · **时长：** 1 小时 30 分钟 · **3 选 2。**

卷二只有一本试卷，含 **2A、2B、2C** 三部分。你只需作答**已修过的两个选修**的全部题目，每部分大约 30 分钟、约 30 分。

## 三个选修对比

| 维度 | **2A · 数据库** | **2B · 网络应用开发** | **2C · 算法与编程** |
|------|---------------|--------------------|------------------|
| 课时 | 38 | 38 | 38 |
| 主要语言 | SQL（SQL-92） | HTML、CSS、JavaScript、**PHP**（2026 起） | Python、C++（Pascal 至 2027） |
| 前置必修模块 | A · 资讯处理 | C · 互联网及其应用 | D · 计算思维与编程 |
| 需要的能力 | 逻辑思维、留意语法 | 设计 + 编码混合 | 算法思维、调试 |
| 难度 | 中等 | 中等 | 偏高 |
| 题目可预测性 | 高 | 中 | 较低（重逻辑） |
| 常见搭配 | 2B（最受欢迎） | 2A 或 2C | 2A 或 2B |

::: tip 我该选哪两个？
- **大多数学生选 2A + 2B**，因为评分模式相对固定，且共用一套 Web + 数据库工具链。
- **数学强**的同学常加 2C，算法题奖励清晰逻辑。
- **编程基础薄弱**不要单独选 2C —— 卷 2C 考算法深度。
:::

## 编程语言对照

| 年份 | 必修 D | 选修 2B | 选修 2C |
|------|--------|---------|---------|
| 2025 DSE | Python（亦接 C++、Pascal） | JavaScript | Python（亦接 C++、Pascal） |
| 2026 DSE | Python（亦接 C++、Pascal） | **加入 PHP** | Python（亦接 C++、Pascal） |
| 2027 DSE | Python（亦接 C++、Pascal） | JavaScript + PHP | Python（亦接 C++、Pascal） |
| 2028 DSE → | **仅 Python 或 C++** | JavaScript + PHP | **仅 Python 或 C++** |
