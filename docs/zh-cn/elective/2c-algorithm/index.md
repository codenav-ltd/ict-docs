# 选修 2C · 算法与编程

> **课时：** 38 · **卷重：** ~12.5% · **语言：** Python、C++（Pascal 至 2027）。

本选修延伸模块 D。除简单循环外，你学算法设计（搜索、排序、归并）、数据结构（栈、队列、链表）、文件处理、事件驱动 / 传感器编程。

## 章节地图

| 章节 | 子题 | 课时 |
|---------|-----------|-------|
| [1 · 编程](./programming/) | 数据类型、嵌套循环、子程序、文件、搜索、排序、结构、测试 | 32 |
| [2 · 现实中的编程](./real-life/) | 传感器、事件、扩展模块 | 6 |

## 为何 2C

- 奖励逻辑思维与整洁代码。
- 技能直接迁移到大学 CS。
- 最「计算机科学」的选修。

## 学习提示

- 从零**每个算法**都练 —— 不只是读。
- 大量用**追踪表**。
- 计时：5 分钟内能写出冒泡排序吗？
- 用 Python REPL 快速试。
