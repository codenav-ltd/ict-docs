## 实证有效的教学策略

| 策略 | 何时 | 为何 |
|------|------|------|
| **IPO 图** | 所有编程课 | 强迫学生写代码前先想清楚 |
| **结对编程** | 编码重的课 | 强者教弱者，双方都受益 |
| **黑板现场追踪表** | 算法课 | 揭开「变量到底干嘛」的神秘 |
| **小型辩论** | 模块 E（伦理） | 调动真实观点，更易记忆 |
| **历届试题「速跑」** | 中六 | 锻炼考试耐力与模式识别 |
| **间隔小测** | 全程 | 对抗艾宾浩斯遗忘曲线 |

## 课堂基础设施建议

- **统一 Python 3.11+**（部分学校仍并行 C++）。
- **每班一个 MySQL 实例**，每生一个 schema；或使用托管多租户工具 [SQL Books](https://sqlbooks.codenav.dev) 让每生有独立数据库 —— 避免「我表被同学删了」。
- **VS Code + Live Share** 协作编程。
- **GitHub Classroom**（或自托管 Gitea）做 SBA 版本控制。
- **公共「常见错误」墙** 每次测验后更新。

## SBA 评分理智检查

1. 学生是否构建了**真能跑**的东西？跑一下。
2. 报告是否**诚实**承认不足？奖励反思。
3. 测试用例是否**多样**（正常/边界/错误）？
4. 设计是否一致（ER 图 = 实际表）？

## 常见学生困难与小干预

| 主题 | 症状 | 干预 |
|------|------|------|
| 伪代码 → Python | 语法错 | 并排模板手册 |
| SQL JOIN | 笛卡尔积混淆 | 文氏图 + 玩具数据集 |
| 二进制补码 | 负数糊涂 | 分步骤练习 × 5 题 |
| HTTP GET/POST | 当成可互换 | 浏览器网络面板演示 |
| 范式化 | 不到 3NF | 从未规范 CSV 出发的工作例 |

## 混合能力班级的差异化教学

- 每个大主题提供**3 档练习卷**：基础、标准、拓展。
- 编程任务给 **脚手架版**（带 TODO）和 **空白版**。
- 中五初做**无成绩诊断小测**找差距。
