# 中四 · 打基础

> 目标：建立词汇和核心技能，不要烧光自己 —— 还有两年。

## 上学期（9–12 月）

| 周 | 重点 | 交付物 |
|----|------|--------|
| 1–3 | 模块 A 简介 + 数据类型 | 进制转换小测 |
| 4–6 | 模块 A 数据组织 + 控制 | Excel 小项目 |
| 7–9 | 模块 C 网络 + 互联网基础 | 家中网络图 |
| 10–12 | 模块 D 问题分析 + 流程图 | 3 题 IPO 工作表 |
| 13 | 期中考 | 模块 A + D 复习 |

### 应装的工具

- Python 3 + 编辑器（推荐 VS Code）
- 浏览器收藏 [SQL Books](https://sqlbooks.codenav.dev) —— 早点习惯读 SQL
- 词汇闪卡笔记（Anki / Quizlet）

## 下学期（1–3 月）

| 周 | 重点 | 交付物 |
|----|------|--------|
| 1–3 | 模块 C 服务 + 网页编写 | 一页静态 HTML |
| 4–6 | 模块 D 算法设计 | 5 题伪代码 |
| 7–9 | 模块 D Python 编程 | 5 个小 Python 程序 |
| 10–12 | 模块 D 测试 | 纸上追踪表 |

### 每周节奏建议

- 1 题编程练习
- 1 条课本 SQL，去 SQL Books 跑
- 50 张词汇闪卡复习

## 暑假前（4–6 月）

| 周 | 重点 | 交付物 |
|----|------|--------|
| 1–3 | 模块 B 硬件 + 系统软件 | DIY PC 预算练习 |
| 4–6 | 模块 E 社会影响 + 伦理 | 500 字伦理意见 |
| 7–10 | 选修品尝（2A → 2B → 2C） | 每个选修 1 题 |
| 11–13 | 期末考 | 全模块 |

### 为什么三个选修都品尝？

中五初你要决定两个选修。提前一周亲手体验各选修，胜过道听途说。

## 暑假建议

- 预读你的两个选修。
- 写一个小 Python 项目（≤ 200 行）。
- 看一段数据库设计入门视频。
- 可选：草拟 SBA 主题，与未来的中五老师讨论。

## 中四结束时的「合格线」

- 30 秒内完成进制转换。
- 看 `SELECT … FROM … WHERE` 能预测输出。
- 能写从文件求和的 Python。
- 能举 3 个知识产权侵权例子。
- 对 2A/2B/2C 有切身感觉。
