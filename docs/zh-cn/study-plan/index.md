## 选修决策树

```
   你喜欢 SQL 和电子表格吗？
   ├── 是 → 选 2A
   │        ├── 也喜欢网页设计吗？
   │        │   └── 是 → 选 2A + 2B（最常见组合）
```

## 三年累积的好习惯

1. **每周写代码**，哪怕 30 分钟。
2. **词汇闪卡** —— 每周 5 个 × 100 周 = 500 词。
3. **历届试题周日** —— 中五下起每月一次。
4. **教会同学** —— 解释一遍就懂了。

::: tip 让练习零阻力
学生不再练习，最大原因是阻力 —— 装工具、修版本、丢文件。从第一天起用浏览器工作台 **[SQL Books](https://sqlbooks.codenav.dev)** 练 SQL，开个标签页就能写 `SELECT`。
:::
