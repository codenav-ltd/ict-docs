# 中六 · 冲刺

> 目标：干净提交 SBA，模拟和真考都达峰。第一个月之后不要学新内容，把已会的锁住。

## 上学期（9–12 月）

| 周 | 重点 | 交付物 |
|----|------|--------|
| 1–3 | 确定 SBA 任务一、二 | 按时提交 |
| 4–6 | 历届试题轮换 | 每周 2 套（1 卷一 + 1 卷二） |
| 7–9 | 弱点定向复习 | 弱点缩到 5 个 |
| 10–12 | 模拟考准备 | 模拟 1 |
| 13 | 校内模拟考 | 完整 DSE 格式 |

### SBA 提交清单

- [ ] 文件夹和格式正确
- [ ] 手册 2–4 页可读
- [ ] 截图带图号
- [ ] 测试用例含正常/边界/错误
- [ ] 自评表签字
- [ ] U 盘 + 云端备份

## 下学期（1–4 月）

| 周 | 重点 | 交付物 |
|----|------|--------|
| 1–3 | 模拟错题分析 | 个人「常见错误」清单 |
| 4–6 | 最终一轮历届试题 | 每周 1 套，计时 |
| 7–9 | 只做间隔记忆 | 词汇、闪卡 |
| 10–12 | 选择性深耕 | 练最弱 3 个 |
| 13 | 末次模拟 | 冲目标等级 |

## 考前 1 周

- **3 天前：** 轻度复习。不学新内容。保证睡眠。
- **前夜：** 看 cheat sheet，准备证件文具，早睡。
- **当天早上：** 清淡早餐，提前到场，深呼吸。

## 考试中

### 卷一（2 小时）

- Section A（MC）：30 分钟。一次快过，不确定的标记，二次重看。
- Section B：80 分钟。先读全部题，做简单的，难的留最后。
- 留 **10 分钟** 检查。

### 卷二（1.5 小时）

- 每选修 45 分钟。
- 看时钟 —— 别让一道难题吃掉另一选修的时间。

## 中六常见陷阱

- **没认真读题**。「Explain」不等于「List」。
- **3 月还学新东西**。坚持已会一半的。
- **跳过 SBA 抛光**。
- **熬通宵**。睡眠比多一套试题宝贵。

## 冲刺末段的「合格线」

- 凭记忆写 CREATE TABLE + INSERT + JOIN。
- 5 分钟内重写冒泡排序或线性查找。
- 10 分钟内画熟悉场景的 ER 图。
- 4 分论 AI 伦理无障碍。

::: tip 用熟悉工具练习
HKEAA 看的是正确逻辑和标准语法，不在乎你用什么工具。日常用 [SQL Books](https://sqlbooks.codenav.dev) 写 SQL，Python 用任意编辑器，PHP 本地起 —— 真考再手写。
:::
