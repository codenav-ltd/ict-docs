# 卷二 · 選修部分總覽

> **比重：** 全科 25% · **時長：** 1 小時 30 分鐘 · **3 選 2。**

卷二隻有一本試卷，含 **2A、2B、2C** 三部分。你只需作答**已修過的兩個選修**的全部題目，每部分大約 30 分鐘、約 30 分。

## 三個選修對比

| 維度 | **2A · 資料庫** | **2B · 網絡應用開發** | **2C · 演演算法與編程** |
|------|---------------|--------------------|------------------|
| 課時 | 38 | 38 | 38 |
| 主要語言 | SQL（SQL-92） | HTML、CSS、JavaScript、**PHP**（2026 起） | Python、C++（Pascal 至 2027） |
| 前置必修模組 | A · 資訊處理 | C · 互聯網及其應用 | D · 計算思維與編程 |
| 需要的能力 | 邏輯思維、留意語法 | 設計 + 編碼混合 | 演算法思維、調試 |
| 難度 | 中等 | 中等 | 偏高 |
| 題目可預測性 | 高 | 中 | 較低（重邏輯） |
| 常見搭配 | 2B（最受歡迎） | 2A 或 2C | 2A 或 2B |

::: tip 我該選哪兩個？
- **大多數學生選 2A + 2B**，因為評分模式相對固定，且共用一套 Web + 資料庫工具鏈。
- **數學強**的同學常加 2C，演算法題獎勵清晰邏輯。
- **編程基礎薄弱**不要單獨選 2C —— 卷 2C 考演算法深度。
:::

## 編程語言對照

| 年份 | 必修 D | 選修 2B | 選修 2C |
|------|--------|---------|---------|
| 2025 DSE | Python（亦接 C++、Pascal） | JavaScript | Python（亦接 C++、Pascal） |
| 2026 DSE | Python（亦接 C++、Pascal） | **加入 PHP** | Python（亦接 C++、Pascal） |
| 2027 DSE | Python（亦接 C++、Pascal） | JavaScript + PHP | Python（亦接 C++、Pascal） |
| 2028 DSE → | **僅 Python 或 C++** | JavaScript + PHP | **僅 Python 或 C++** |
