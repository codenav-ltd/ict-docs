# 選修 2A · 資料庫

> **課時：** 38 · **卷重：** ~12.5% · **主要語言：** SQL (SQL-92)。

本選修延伸模組 A。你學關係型理論、用 ER 圖設計資料庫、規範化模式、寫 SQL 查詢與修改資料庫。

## 2A 為何熱門

- 評分穩定 —— 題型套路成熟。
- 與 SBA 緊密 —— 項目裏幾乎一定要用 SQL。
- SQL 是遠超考試的可遷移技能。

## 學習提示

- **每個概念都要寫程式碼。** 讀 SQL 是寫它一半的速度。打開 [SQL Books](https://sqlbooks.codenav.dev) 與本站並排。
- 為 SQL **保留字列表**做閃卡。
- 給每個示例情境**手畫 ER 圖** —— 哪怕草圖。
