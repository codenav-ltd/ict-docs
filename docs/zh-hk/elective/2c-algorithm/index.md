# 選修 2C · 演算法與編程

> **課時：** 38 · **卷重：** ~12.5% · **語言：** Python、C++（Pascal 至 2027）。

本選修延伸模組 D。除簡單迴圈外，你學演算法設計（搜索、排序、歸併）、資料結構（棧、隊列、鏈表）、文件處理、事件驅動 / 傳感器編程。

## 為何 2C

- 獎勵邏輯思維與整潔代碼。
- 技能直接遷移到大學 CS。
- 最「計算機科學」的選修。

## 學習提示

- 從零**每個演算法**都練 —— 不只是讀。
- 大量用**追蹤表**。
- 計時：5 分鐘內能寫出冒泡排序嗎？
- 用 Python REPL 快速試。
