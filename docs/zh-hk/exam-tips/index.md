# 應試策略總覽

> 4 分與 5* 的差距，往往不是「知識更多」，而是**技巧、時間紀律、避免自損**。

## 通用建議（兩份卷子都適用）

1. **題目讀兩遍**。圈出命令動詞（解釋、識別、寫、設計）。
2. **寫前先規劃**。30 秒結構勝過 5 分鐘漫談。
3. **分值就是你的碼錶**。1 分題 1 分鐘，不是 5 分鐘。
4. **不會就寫你會的**。部分分也是分。
5. **多畫圖**。清晰的圖比文字得分更快。

## 1-2-3 覆盤法

每套歷屆試題後：

1. **誠實自評**。
2. **標記每個錯題** 的模組 + 子話題。
3. **整理「弱點閃卡」**。

這副閃卡是中六最值錢的溫習資產。
