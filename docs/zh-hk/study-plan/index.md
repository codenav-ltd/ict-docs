## 選修決策樹

```
   你喜歡 SQL 和電子表格嗎？
   ├── 是 → 選 2A
   │        ├── 也喜歡網頁設計嗎？
   │        │   └── 是 → 選 2A + 2B（最常見組合）
   │        └── 也熱愛演算法嗎？
   │            └── 是 → 選 2A + 2C
   └── 否 → 
        你喜歡邏輯和編程嗎？
        ├── 是 → 選 2C
        │       └── 喜歡網頁則加 2B
        └── 否 → 重新考慮 ICT 選修是否適合你
```

## 三年累積的好習慣

1. **每週寫程式碼**，哪怕 30 分鐘。
2. **詞彙閃卡** —— 每週 5 個 × 100 周 = 500 詞。
3. **歷屆試題週日** —— 中五下起每月一次。
4. **教會同學** —— 解釋一遍就懂了。

::: tip 讓練習零阻力
學生不再練習，最大原因是阻力 —— 裝工具、修版本、丟文件。從第一天起用瀏覽器工作台 **[SQL Books](https://sqlbooks.codenav.dev)** 練 SQL，開個標籤頁就能寫 `SELECT`。
:::
