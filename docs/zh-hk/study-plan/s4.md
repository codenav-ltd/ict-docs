# 中四 · 打基礎

> 目標：建立詞彙和核心技能，不要燒光自己 —— 還有兩年。

## 上學期（9–12 月）

| 周 | 重點 | 交付物 |
|----|------|--------|
| 1–3 | 模組 A 簡介 + 資料類型 | 進制轉換小測 |
| 4–6 | 模組 A 資料組織 + 控制 | Excel 小項目 |
| 7–9 | 模組 C 網絡 + 互聯網基礎 | 家中網絡圖 |

### 應裝的工具

- Python 3 + 編輯器（推薦 VS Code）
- 瀏覽器收藏 [SQL Books](https://sqlbooks.codenav.dev) —— 早點習慣讀 SQL
- 詞彙閃卡筆記（Anki / Quizlet）

### 為什麼三個選修都品嚐？

中五初你要決定兩個選修。提前一週親手體驗各選修，勝過道聽途説。

## 暑假建議

- 預讀你的兩個選修。
- 寫一個小 Python 項目（≤ 200 行）。
- 看一段資料庫設計入門視頻。
- 可選：草擬 SBA 主題，與未來的中五老師討論。

## 中四結束時的「合格線」

- 30 秒內完成進制轉換。
- 看 `SELECT … FROM … WHERE` 能預測輸出。
- 能寫從文件求和的 Python。
- 能舉 3 個知識產權侵權例子。
- 對 2A/2B/2C 有切身感覺。
