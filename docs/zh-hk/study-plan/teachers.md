## 實證有效的教學策略

| 策略 | 何時 | 為何 |
|------|------|------|
| **IPO 圖** | 所有編程課 | 強迫學生寫程式碼前先想清楚 |
| **結對編程** | 編碼重的課 | 強者教弱者，雙方都受益 |
| **黑板現場追蹤表** | 演算法課 | 揭開「變數到底幹嘛」的神秘 |

| **歷屆試題「速跑」** | 中六 | 鍛鍊考試耐力與模式識別 |
| **間隔小測** | 全程 | 對抗艾賓浩斯遺忘曲線 |

## 課堂基礎設施建議

- **統一 Python 3.11+**（部分學校仍並行 C++）。
- **每班一個 MySQL 實例**，每生一個 schema；或使用託管多租户工具 [SQL Books](https://sqlbooks.codenav.dev) 讓每生有獨立資料庫 —— 避免「我表被同學刪了」。
- **VS Code + Live Share** 協作編程。
- **GitHub Classroom**（或自託管 Gitea）做 SBA 版本控制。
- **公共「常見錯誤」牆** 每次測驗後更新。

## SBA 評分理智檢查

1. 學生是否構建了**真能跑**的東西？跑一下。
2. 報告是否**誠實**承認不足？獎勵反思。
3. 測試用例是否**多樣**（正常/邊界/錯誤）？
4. 設計是否一致（ER 圖 = 實際表）？

## 常見學生困難與小干預

| 主題 | 症狀 | 干預 |
|------|------|------|
| 偽程式碼 → Python | 語法錯 | 並排模板手冊 |
| SQL JOIN | 笛卡爾積混淆 | 文氏圖 + 玩具資料集 |
| 二進制補碼 | 負數糊塗 | 分步驟練習 × 5 題 |
| HTTP GET/POST | 當成可互換 | 瀏覽器網絡面板演示 |
| 範式化 | 不到 3NF | 從未規範 CSV 出發的工作例 |

## 混合能力班級的差異化教學

- 每個大主題提供**3 檔練習卷**：基礎、標準、拓展。
- 編程任務給 **腳手架版**（帶 TODO）和 **空白版**。
- 中五初做**無成績診斷小測**找差距。
