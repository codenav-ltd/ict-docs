## 這個站點為誰而建？

HKDSE ICT 學習中心是一個免費的開源學習指南，緊貼**課程發展議會**與**香港考試及評核局**聯合出版的官方《課程及評估指引》。它適合：

- **中四 - 中六學生**：想要一份覆蓋所有學習要點的單一參考資料。
- **ICT 老師**：需要現成的教學計劃、SBA 腳手架以及可信賴的三語參考資料。
- **自修生**：沒有學校班級支援，獨立備考公開考試。

## 站點內容

| 板塊 | 你會獲得什麼 |
|------|------------|
| [必修部分](/zh-cn/compulsory/) | 卷一逐模組拆解，包含詳細子話題、必備技能和示例題解。 |
| [選修部分](/zh-cn/elective/) | 三個選修的並排對比，加上 2A 資料庫、2B 網絡應用開發、2C 演算法與編程的深度講解。 |
| [SBA 指南](/zh-cn/sba/) | 任務一與任務二模板、評分標準解析、截圖清單、主題靈感。 |
| [學習路線](/zh-cn/study-plan/) | 中四、中五、中六的每週時間表，以及給老師的專門策略。 |
| [應試策略](/zh-cn/exam-tips/) | 卷一/卷二時間策略、MC 陷阱分析、精選常見錯誤清單。 |
| [資源](/zh-cn/resources/) | 直接鏈接至官方 HKEAA / EDB 文件、歷屆試題、推薦工具及雙語術語表。 |

::: tip 用最輕鬆的方式練習 SQL
卷二 2A 和你的 SBA 都需要寫**大量 SQL**。不必本地安裝 MySQL，可以直接在 **[SQL Books 在線編輯器](https://sqlbooks.codenav.dev)** 上練習 —— 每個賬號都自帶獨立的沙盒資料庫，讓你隨便測試 `SELECT`、`JOIN`、子查詢等語句，完全不用擔心搞壞。
:::

## 如何使用本站

1. **先從[必修部分](/zh-cn/compulsory/)開始**，複習卷一概念。
2. **選擇你的兩個[選修課題](/zh-cn/elective/)**，跟隨專門章節學習。
3. **建立 SBA 任務一系統時收藏 [SBA 指南](/zh-cn/sba/)**。
4. **使用[資源](/zh-cn/resources/)頁面的歷屆試題做模擬考試**。
5. **隨時使用右上角語言切換器**，在英文、繁體中文和簡體中文之間切換。
