## 全局圖

軟件有兩種：**系統軟件**（讓機器自身運轉）和**應用軟件**（用户真正用的）。

```
┌────────────────────────────┐
│ 應用軟件                    │  Word、Chrome、遊戲…
├────────────────────────────┤
│ 系統軟件                    │  OS + 實用程序 + 驅動
├────────────────────────────┤
│ 硬件                        │  CPU、RAM、磁碟…
└────────────────────────────┘
```

系統軟件讓燈長亮。沒有 OS、驅動、實用程序，應用軟件跑不起來。
