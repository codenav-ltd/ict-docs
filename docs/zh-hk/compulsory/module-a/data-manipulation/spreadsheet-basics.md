# 4.1 · 電子表格基礎

> **目標：** 説出電子表格各部分名稱、寫基本公式、用篩選 / 排序 / 搜索。

## 電子表格解剖

```
   A         B         C         D
1  Name      Class     Score     Grade
2  Alice     F.4A      86        =IF(...)
3  Bob       F.4A      72        =IF(...)
4  Carol     F.4B      91        =IF(...)
```

| 術語 | 含義 |
|------|---------|
| **Workbook 工作簿** | 整個文件（`.xlsx`） |
| **Worksheet 工作表（sheet / tab）** | 工作簿內的一頁 |
| **Column 列** | 字母 A、B、C、… |
| **Row 行** | 數字 1、2、3、… |
| **Cell 單元格** | 一個交點（如 `B3`） |
| **Range 區域** | 矩形單元格組（如 `A2:D10`） |
| **Formula 公式** | 以 `=` 開頭的表達式 |
| **Function 函式** | 預定義公式（如 `SUM`、`AVERAGE`） |

## 寫你的第一個公式

每個公式都以 `=` 開始。運算符：

| 運算符 | 含義 | 例子 |
|----------|---------|---------|
| `+` `-` `*` `/` | 算術 | `=A1+B1` |
| `^` | 冪 | `=2^10` |
| `&` | 拼接文本 | `="Hello "&A1` |
| `=` `<>` `>` `<` `>=` `<=` | 比較 | `=A1>=50` |

```
A1: 50
A2: 30
A3: =A1+A2    →  80
A4: =A1*2     →  100
A5: =SUM(A1:A4) → 80+50+30+100 = 260
```

## 三個必備操作

### 1 · 排序

| 之前 | 按 Score 降序排後 |
|--------|---------------------------|
| Alice 86 | Carol 91 |
| Bob 72 | Alice 86 |
| Carol 91 | Bob 72 |

可按**單**或**多個標準**排（如先 Class 再 Score）。

### 2 · 篩選

只顯示符合條件的行，其他行臨時隱藏。

| 篩選 | 結果 |
|--------|--------|
| Class = F.4A | Alice 與 Bob 可見；Carol 隱藏 |
| Score ≥ 80 | Alice 與 Carol 可見 |

### 3 · 搜索 / 查找替換

- 查找：在工作表中定位一個值。
- 查找替換：批量把一個值換成另一個。

## 學生常見錯誤

- 公式開頭忘了 `=`。
- 用 `*` 而不輸單元格引用：`=A1 * 2` 可以，但 `=A1 * B1` 兩個引用都要寫。
- 排序時沒選**整張表** —— 把名字和分數拆開。
- 不先清除之前的篩選就再篩 → 部分視圖令人困惑。

## 實例 · 等級計算器

在 `D2` 輸入：

```text
=IF(C2 >= 80, "A", IF(C2 >= 70, "B", IF(C2 >= 60, "C", "F")))
```

下拉到 `D3:D4` 即可給每位學生評級。

## 練習活動

新建上面所示的工作表。然後：

1. 按 score 降序排學生。
2. 篩只看 F.4A。
3. 加一列「Average」顯示班級平均。
4. 用條件格式把低於 60 的分數標紅。

## 關鍵要點

- 單元格、行、列、區域 —— 術語要熟。
- 每個公式以 `=` 開頭。
- 排序、篩選、查找是第一天就要會的三個操作。
