# 模組 A · 資訊處理

> **課時：** 37 · **考核：** 卷一（A 部分 & B 部分） · **銜接：** 選修 2A 資料庫

## 你將學到什麼

學完本模組，你應該能夠：

- 識別**資訊系統**的**組成與流程**。
- 區分**資料與資訊**，並解釋為何這種區分重要。
- 自如地在**十進制、二進制、十六進制**之間轉換數字。
- 進行**二進制運算**，包括二的補碼。
- 認識各種**字符編碼**（ASCII、Big-5、GB、Unicode），並選對適用場景。
- 決定哪種**多媒體文件格式**適合特定用途。
- 用**電子表格**做篩選、排序、透視和資料分析。
- 用 **DBMS** 建立簡單資料庫、表單、查詢和報表。
- 讀懂並解釋簡單的 **SQL** 語句。

## 建議學習順序

1. **先看第 1 章** —— 它教你後續章節默認你已掌握的術語。
2. **接着第 3 章** —— 數制熟練度是開啓其他模組的鑰匙。
3. **第 2 章** —— 短小且概念性強，容易拿分。
4. **第 4 章** —— 子題最大，是通往選修 2A 的橋樑。

## 典型題目問法

- 卷一 A 部分：2–4 道選擇題，考數制轉換、validation/verification、文件格式選擇。
- 卷一 B 部分：通常一道結構題，情境（學校辦公室、運動會、小生意）會讓你把電子表格邏輯、SQL 閲讀和校驗規則混合使用。

::: tip 一遇 SQL 就動手練
本模組第一次看到 `SELECT … FROM …` 時，立刻打開 **[SQL Books](https://sqlbooks.codenav.dev)** 並排放。把例子粘進去執行，改一句再執行。被動地讀 SQL **比**動手跑慢得多。
:::
