# 3.2 · 變數與運算符

> **目標：** 聲明變數、用對運算符、遵守命名規範。

## 變數

變數是給一個值起的名字。

```python
age = 17
name = "Alice"
height = 1.65
is_student = True
```

Python 推斷類型 —— 無需顯式聲明。

## 命名規則

- 字母、數字、下劃線；必須以字母或 `_` 起。
- 區分大小寫：`Age` ≠ `age`。
- 別用 Python 關鍵字（`if`、`for`、`class`…）。
- **慣例**：變數與函式用 `snake_case`，常量用 `UPPER_CASE`。

好：

```python
average_height = 165
PI = 3.14159
```

壞：

```python
a = 165          # 'a' 是什麼意思？
PiValue = 3.14   # 風格混雜
```

## 運算符速查

### 算術

| 運算符 | 含義 | 例子 | 結果 |
|----------|---------|---------|--------|
| `+` | 加 | `3+2` | 5 |
| `-` | 減 | `3-2` | 1 |
| `*` | 乘 | `3*2` | 6 |
| `/` | 真除 | `5/2` | 2.5 |
| `//` | 地板除 | `5//2` | 2 |
| `%` | 取餘 | `5%2` | 1 |
| `**` | 冪 | `2**10` | 1024 |

### 比較（關係）

| 運算符 | 含義 |
|----------|---------|
| `==` | 等於 |
| `!=` | 不等 |
| `>` `<` | 大 / 小 |
| `>=` `<=` | 大等於 / 小等於 |

::: warning Python vs HKEAA 偽程式碼
HKEAA 偽程式碼用 `=` 比較、`←` 賦值。Python 用 `==` 比較、`=` 賦值。答案別混。
:::

### 布林（邏輯）

| 運算符 | 含義 |
|----------|---------|
| `and` | 兩個操作數都 True 時為 True |
| `or` | 至少一個 True 時為 True |
| `not` | 反轉布林 |

### 賦值快捷

| 快捷 | 含義 |
|----------|-------|
| `x += 5` | `x = x + 5` |
| `x -= 3` | `x = x - 3` |
| `x *= 2` | `x = x * 2` |

## 運算符優先級（高 → 低）

1. `**`
2. `*` `/` `//` `%`
3. `+` `-`
4. 比較 `<` `<=` `>` `>=` `==` `!=`
5. `not`
6. `and`
7. `or`

不確定就加括號。

## 實例

```python
total = 0
for i in range(1, 11):
    total += i              # 1+2+...+10
print(total)                # 55

a, b = 7, 3
print(a // b, a % b)        # 2 1   (商, 餘)

x = 5
print(x ** 2)               # 25
print((x + 3) * 2)          # 16
```

## 學生常見錯誤

- 混淆 `=`（賦值）與 `==`（比較）。
- 忘了 `5 / 2 = 2.5`（Python 3 真除法）；整除用 `//`。
- 用 `&` 與 `|`（位運算）代替 `and` / `or`（布林）。
- 用變數名遮蔽內建（`list`、`sum`、`str`）。

## 考試式題目

> **題（4 分）：** 逐行追蹤 `result` 的值：
>
> ```python
> x = 7
> y = 3
> result = (x % y) * 2 + (x // y) ** 2
> ```

**參考答案：**

- `x % y = 1`
- `(1) * 2 = 2`
- `x // y = 2`
- `2 ** 2 = 4`
- `result = 2 + 4 = 6`

## 關鍵要點

- 名字重要 —— 取有意義的。
- 知道你的運算符及優先級。
- Python 的 `=` 是賦值；`==` 是比較。
