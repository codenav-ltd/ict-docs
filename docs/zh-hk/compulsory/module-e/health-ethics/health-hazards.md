# 2.1 · ICT 健康危害

> **目標：** 識別長時間使用 ICT 的生理與心理危害。

## 生理危害

| 危害 | 原因 |
|--------|-------|
| **重複性勞損 (RSI)** | 反覆鍵鼠動作 |
| **腕管綜合徵** | 手腕神經壓迫 |
| **眼疲勞 / 計算機視覺綜合徵** | 長屏幕時間、藍光、低對比 |
| **頸 / 背 / 肩痛** | 不良姿勢、椅子設計差 |
| **乾眼** | 盯屏幕時眨眼減少 |
| **頭痛** | 眼疲勞、脱水、光線差 |
| **聽力損傷** | 長期高音量用耳機 |
| **肥胖** | 久坐屏幕時間 |
| **睡眠紊亂** | 藍光 + 深夜使用擾亂晝夜節律 |

## 心理與社交危害

| 危害 | 原因 |
|--------|-------|
| **焦慮** | 不斷通知、社交比較 |
| **抑鬱** | 過度社交媒體 |
| **網絡欺凌** | 在線匿名 |
| **成癮** | 遊戲、社交媒體獎勵迴圈 |
| **FOMO**（錯失恐懼） | 實時看見他人生活 |
| **孤立** | 在線取代線下關係 |

## 快速建議

- **20-20-20 法則** —— 每 20 分鐘看 20 英尺（6 米）外 20 秒。
- **每小時微休息** —— 站立、伸展、眨眼。
- 耳機**音量限制**（60% / 60 分鐘是一個指南）。
- **睡前屏幕宵禁** —— 至少睡前 30 分鐘。
- **關通知**減焦慮觸發。

## 香港情境

- 不少港生報告每日 > 6 小時屏幕時間。
- 香港衞生署出版學生人體工學指南。
- 政府智慧城市藍圖提到數碼健康。

## 考試式題目

> **題（5 分）：** 描述長時間用 ICT 的三種健康危害，並各建議一項預防措施。

**參考答案：**

1. 連續打字致 **RSI** → 每 30 分鐘微休息；做手腕伸展；用人體工學鍵盤。
2. 屏暴露致**眼疲勞** → 20-20-20 法則、調亮度、增大字號。
3. 深夜屏使用致**睡眠紊亂** → 開夜間模式（暖色），睡前 30 分鐘避免屏幕。

## 關鍵要點

- 危害有生理（RSI、眼疲勞）也有心理（焦慮、成癮）。
- 簡單習慣防多數問題。
