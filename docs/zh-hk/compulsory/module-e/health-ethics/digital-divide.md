# 2.3 · 數碼鴻溝

> **目標：** 解釋數碼鴻溝並識別受影響羣體。

## 定義

**數碼鴻溝**是能接觸並有效使用 ICT 的人與不能的人之間的差距。

## 鴻溝的維度

| 維度 | 問題 |
|-----------|-------|
| **收入** | 設備成本、互聯網訂閲 |
| **地理** | 偏遠地區連接差 |
| **年齡** | 長者不熟數字介面 |
| **障礙** | App 不可及、缺輔助技術 |
| **性別** | 歷史上科技行業代表不足 |
| **教育** | 識字低 → 難用在線服務 |
| **語言** | 在線內容以英文為大宗 |
| **技能** | 「有設備但不會高效用」 |

## 後果

- **工作機會減少**，越來越多崗位要求數字技能。
- **政府服務有限**，因服務搬上線。
- **教育不平等** —— 無設備的學生掉隊。
- 不用通訊 App 的長者**社交孤立**。

## 香港情境

- 整體互聯網普及高，但長者與低收入羣體代表不足。
- 學校為有需學生提供筆電。
- 公共圖書館提供免費 Wi-Fi 與電腦課。
- 長者上網基金會為長者辦課程。

## 縮小鴻溝

| 策略 | 例子 |
|----------|---------|
| **補貼設備** | 疫情期政府筆電貸 |
| **免費公共 Wi-Fi** | 康文署圖書館、港鐵站 |
| **數碼素養課** | 長者中心、NGO |
| **可及設計** | 大字、語音命令 |
| **多語介面** | 翻譯 App 與網站 |

## 考試式題目

> **題（5 分）：** 討論數碼鴻溝並提議兩項香港可採取以縮小的措施。

**參考答案：**

數碼鴻溝指能接觸並有效使用 ICT 的個人與不能的人之間的差距。在香港它格外影響**長者**（不熟數字介面）、**低收入家庭**（設備與寬帶成本）和**殘障人士**（缺可及內容）。後果包括被在線政府服務排除、工作機會減少、教育不平等。

兩項措施：

1. **為低收入學生補貼設備 + 寬帶** —— 延續疫情筆電貸計劃並提供免費或低價互聯網，讓孩子能做在線作業。
2. **公共圖書館與長者中心**為長者辦**免費數碼素養課**，配雙語（粵語 / 英語 / 普通話）指導，教必備技能如網銀、電子政府、與家人視訊。

## 關鍵要點

- 多維度 —— 不止收入。
- 影響特定香港羣體。
- 縮小需設備、連接、**且**技能。
