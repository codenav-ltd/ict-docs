# 2.2 · 人體工學

> **目標：** 描述符合人體工學的工作站並解釋它如何減少損傷。

## 人體工學工作站清單

| 要素 | 建議 |
|---------|----------------|
| **椅** | 可調高度；腰部支撐；扶手；腳平放地面（或腳踏） |
| **顯示器** | 屏頂在眼平；離眼 50–70 cm；抗眩光霧面 |
| **鍵盤** | 平或微傾；腕直，不上彎 |
| **鼠標** | 貼近鍵盤；手同高度；支撐腕墊 |
| **桌** | 穩定，夠大放鍵鼠 + 文件 |
| **照明** | 間接；屏無反光；與自然光互補 |
| **空氣** | 温度舒適；不太乾 |
| **噪音** | 60 dB 以下；需要時降噪耳機 |

## 姿勢圖

```
  眼平 ------ 屏頂
              ↘
                ── 屏 50-70 cm
              
   肘 90°
   腕直
   背挺（微後倚 OK）
   髖齊 或略高於膝
   腳平放地面（或腳踏）
```

## 軟件人體工學

課程指引提到**軟件人體工學** —— 減少努力與挫敗的設計：

- 跨屏**一致 UI**。
- **熟悉慣例**（`Ctrl-S` 保存）。
- **明確錯誤資訊**。
- **無障礙**（屏幕閲讀器支援、對比、字號縮放）。
- 常用動作**少點擊**。

好軟件人體工學 → 少出錯、做得快、壓力小。

## 常見壞習慣

- 筆電放腿上（頸太低、腕太高）。
- 單一大屏高度錯且無獨立鍵盤。
- 屏後亮窗 → 眼疲勞。
- 雜亂桌面強迫尷尬姿勢。

## 忽視人體工學的代價

- 長期 RSI / 腕管可能要手術。
- 持續小痛降低生產力。
- 缺勤增多。

## 考試式題目

> **題（4 分）：** 為每天用筆電在飯桌上 4 小時上網課的學生建議四項人體工學改進。

**參考答案：**

1. 用支架**抬高筆電**並外接鍵鼠，讓屏幕在眼平且手腕自然。
2. **用可調椅**帶背支撐與腳踏，替換硬飯桌椅。
3. **改善照明** —— 把筆電與窗垂直放避眩光；晚學習用枱燈。
4. **每 30–45 分鐘休 5 分鐘**站、伸展、跟 20-20-20 法則。

## 關鍵要點

- 調椅、屏、鍵、鼠、燈。
- 軟件設計也影響人體工學。
- 小改動，長遠健康收益大。
