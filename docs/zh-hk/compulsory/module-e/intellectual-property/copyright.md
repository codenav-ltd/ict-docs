# 3.1 · 版權基礎

> **目標：** 定義版權並區分其與相關權利。

## 版權保護什麼

**版權**是**原創**作品的創作者控制其**複製、發行、改編、表演與展示**的法律權利。

涵蓋的作品：

- 文學作品（書、文章、源程式碼）
- 音樂作品
- 電影、電視節目、錄製音頻
- 美術作品（繪畫、相片、圖形）
- 建築設計
- 計算機軟件

版權**不**保護的：

- 想法（只保護想法的*表達*）
- 事實（二戰日期）
- 政府法律與司法判決

## 版權如何取得

多數國家（含香港），作品一旦以有形形式創作即**自動**取得版權 —— 無需註冊。© 符號與日期可選但是慣例。

## 期限

- 香港，文學、戲劇、音樂、美術作品版權一般為**作者壽命 + 50 年**。
- 期滿後作品進入**公有領域**，可自由使用。

## 相關權利

| 權利 | 保護 |
|-------|----------|
| **商標** | 徽標、品牌 |
| **專利** | 發明（須新穎且實用） |
| **商業秘密** | 機密商業資訊（如配方） |
| **設計權** | 工業設計（形狀、外觀） |
| **人格權** | 作者被署名與作品不被歪曲的權利 |

## 合理使用 / 公平處理

多數司法管轄區允許有限地使用版權作品無需許可，用於：

- 教育
- 新聞報道
- 評論 / 審查
- 戲仿 / 諷刺
- 研究

香港在《版權條例》(Cap. 528) 下用術語**公平處理 (fair dealing)**。

## 例子

| 行為 | 可能狀態 |
|--------|---------------|
| 引用教材兩句並附引用 | 公平處理 |
| 在電影院錄電影並上載 | 侵權 |
| 未授權採樣歌曲 | 通常侵權 |
| 分享名人 meme 圖 | 常被容忍；法理複雜 |
| 按許可條款用開源庫並署名 | 允許 |

## 香港法律背景

- **《版權條例》(Cap. 528)** 管港版權。
- **商業規模**侵權刑事責任（賣盜版 DVD）。
- 嚴重終端用户盜版可致罰款或監禁。

## 考試式題目

> **題（4 分）：** 解釋版權保護什麼並列三個涵蓋作品的例子。

**參考答案：**

版權保護文學、音樂、美術與數字作品的**原創表達**，禁未經許可複製、發行、改編或表演。它不保護想法，只保護想法的表達方式。三例：出版的小説、錄製的歌曲、計算機源程式碼。

## 關鍵要點

- 版權保護表達，不保護想法。
- 創作即自動；無需註冊。
- 期限：通常作者壽命 + 50 年（香港）。
